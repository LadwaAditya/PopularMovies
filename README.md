An app to keep track with movies

# Libraries used


* Android Design Library
* Butter Knife
* Glide
* Glide Pallet
* Retrofit
* OkHttp
* RxJava
* RxAndroid
* Stetho


# How to use this
### Add your API key

```bash
 <string name="api_key">Your API key</string>
```
